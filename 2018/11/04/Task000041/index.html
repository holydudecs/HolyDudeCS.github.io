Q2 Answer:

1)lsof: lsof means List Open Files by each process.

lsof is the command in Linux for displaying opened files for each process. it displays command, user name, pid, type of device and size of file or process etc.

lsof syntax: lsof +［options］+ filename

  • a: list open files exist process;

  • c < process name > : list open files specified process;

-g: list the details of the GID process;

  • d < file >: list takes up the file number process;

  • d < directory >: list file directory is opened;

  • D < directory >: recursive list file directory is opened;

N < directory >: list the files that use NFS;

-I < condition >: list the qualifying processes. (4, 6, protocol, port, @ip)

-p< process number >: lists the files opened by the specified process number;

-u: list the details of the UID process;

-h: display help information;

  • v: shows version information.
